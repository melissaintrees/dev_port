NPR’s The Record Web Scraper

NPR’s “The Record” Web Scraper is a full stack app that takes articles from existing website and saves info associated with each article to a database that is hosted on the app. Users can add notes to the articles. The result is an attractive interface and useful tool where music fans can track and store their thoughts on latest industry trends.

Built with: Bootstrap, HTML, CSS, Node, Express, Request, Mongoose, Handlebars, Sketch
